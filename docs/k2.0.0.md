<br/>

>[!WARNING|style: flat|label: 简要说明 ]
>
>(`NLog`) 的日志路由规则`rules`是有序集合，<span style='color:red'>按顺序依次处理</span> [<span style='color:#008B00'>[👓 官方说明 ]</span>](https://github.com/NLog/NLog/wiki/Configuration-file#rules ':target=_blank')
>
>- <span style='color:red'>匹配规则：从第一条规则开始，逐条判断日志类别(`name`) + [ 日志级别`minLevel/maxLevel`是否匹配 ]</span>
>
>- <span style='color:red'>匹配成功：日志会被路由到其 [ 指定的目标`target`]</span>
>
>   <span style='color:red'>[ 默认情况下，日志会继续匹配后续规则，直到所有规则都处理完，或者遇到设置了`final: true`的规则为止，此时后续规则将被忽略 ]</span>
>
>
>
>
>| <span style='color:Blue'>[`logger`配置项 ]</span> |                                                              |
>| ------------------------------------------------- | ------------------------------------------------------------ |
>| `name`                                            | 匹配`Category`支持通配符(`* 和 ?`)                           |
>| `minlevel, maxlevel`                              | 匹配日志范围的 [ 最小级别  + 最大级别 ]                      |
>| `level`                                           | 匹配指定日志级别                                             |
>| `levels`                                          | 匹配指定日志级别组 [ 通过`,`分割 ]                           |
>| -                                                 |                                                              |
>| `writeTo`                                         | <span style='color:red'>[ 日志输出 - 多目标由逗号分隔 ]</span> |
>| `final`                                           | <span style='color:red'>[ 规则终止`return`] [ 推荐使用 ]</span> |
>| `enabled`                                         | <span style='color:Blue'>[ 禁用规则`enabled = true`]</span>  |
>| `finalMinLevel`                                   | [ 等效于`minlevel`+`final`] [ 不记录任何日志`finalMinLevel="Off"`] [<span style='color:#008B00'>[👓 官方说明 ]</span>](https://github.com/NLog/NLog/wiki/Logging-Rules-FinalMinLevel  ':target=_blank')<br/><span style='color:red'>[ 注意：不推荐，实际会被各种分流 ]</span> |
>
>⚠ <span style='color:red'>[ 日志规则顺序是非常重要的 ]</span>
>
><br/>

#### **XML**

```xm
<rules>

     <logger name="Microsoft.Hosting.Lifetime*" minlevel="Info" writeTo="console" final=true /> 
     <!-- [ Microsoft.* 规则兜底 ] -->
     <logger name="Microsoft.*" minlevel="Warn" writeTo="console" final=true />
     
     <logger name="PM2.Exceptions.*" minlevel="Info" writeTo="requestTarget" final=true />
     <!-- [ PM2.* 规则兜底 ] -->
     <logger name="PM2.*" minlevel="Info" writeTo="Default_Console" final=true />
     
     <!-- [ 总体规则兜底 ] -->
     <logger name="*" minLevel="Info" writeTo="console" />
    
</rules>


```



#### **JSON**

```json
"rules":[
    
     { "logger":"Microsoft.Hosting.Lifetime*", "minlevel":"Info", "writeTo":"Default_Console", "final": true },
     { "logger":"Microsoft.*", "minlevel":"Warn", "writeTo":"Default_Console", "final": true },
     
     { "logger":"PM2.Exceptions.*", "minlevel":"Info", "writeTo":"Default_Console", "final": true },
     { "logger":"PM2.*", "minlevel":"Info", "writeTo":"Default_Console", "final": true },
     
     { "logger":"*", "minLevel":"Info", "writeTo":"Default_Console" }
    
]


```
